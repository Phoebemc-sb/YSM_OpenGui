# 🚀 YSMParseropen-Gui (GUI Plugin Edition)

这是一个专为 [OpenYSM/YSMParser](https://github.com/OpenYSM/YSMParser) 开发的**独立图形化前端插件**。

> **重要声明**：本项目**仅包含外部 GUI 界面代码**。为了完全符合开源合规性并尊重原作者版权，本项目的任何源码中**均不包含、不涉及、亦不提供**任何关于 YSM 模型的解密、反编译、逆向工程或核心解析逻辑。

---

## 📌 项目定位

本项目是一个**纯粹的壳程序（Wrapper / Plugin）**。
它的作用是为原作者编写的命令行工具提供一个现代化的、可视化的操作界面。所有的核心解析与解密动作，均需要用户自行下载原版核心并调用执行。

## ✨ 插件特性

- **现代深色 UI**：采用基于 GitHub 风格的极客黑皮肤，美观且护眼。
- **一键路径适配**：针对 YSM V3 版本的目录结构进行智能优化，自动处理 Windows 路径。
- **实时日志流**：无缝捕获核心程序的标准输出，在界面中直观展示解析进度。
- **快捷直通**：界面左下角集成一键直通本仓库的开源按钮，方便用户提交 Issue 或点亮 Star。

## ⚠️ 核心合规性与免责声明 (Crucial Compliance & Disclaimer)

1. **源码无解密逻辑**：本仓库的 Python 源码（`main.py`）仅负责界面绘制、文件选择器调用以及进程拉起（`subprocess`）。我们没有、也绝不包含任何破解或阅读 YSM 加密数据的代码。
